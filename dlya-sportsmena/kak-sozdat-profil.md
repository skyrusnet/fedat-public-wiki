# Как создать спортивное портфолио

**Актуально сейчас (пилот):** заявка на [profile.fedat.ru](https://profile.fedat.ru) → **ссылка для редактирования** → заполнение страницы портфолио → **публикация** → ссылка для друзей и тренера.

> **Важно:** адрес страницы портфолио (`…/index.php?id=…`) **приходит вам персонально** после заявки — не ищите его вручную на сайте. Не передавайте никому ссылку с параметром `edit=` — это доступ к редактированию.

---

## Шаг 1. Заявка на profile.fedat.ru

1. Открой [https://profile.fedat.ru](https://profile.fedat.ru) на телефоне или компьютере.
2. Заполни форму: **имя**, **вид спорта**, **роль** (спортсмен / родитель / тренер), **контакт** (телефон или Telegram).
3. При желании выбери пакет (Start / Pro / Elite) — на этапе пилота главное — получить портфолио.
4. Отправь заявку.

После успешной отправки на странице может сразу появиться кнопка **«Перейти в своё портфолио»** — это ваша **ссылка для редактирования**. Скопируйте и сохраните её.

**Если ссылки на экране нет** — профиль всё равно создаётся; ссылку пришлёт менеджер (мессенджер, почта). Срок — по загрузке команды, обычно в течение рабочего дня.

---

## Шаг 2. Открыть ссылку для редактирования

1. Перейди по ссылке из заявки или от менеджера. В адресе будет параметр **`edit=`** — одноразовый ключ доступа.
2. Браузер может **убрать `edit=` из адресной строки** после загрузки — это нормально: доступ сохраняется в этом браузере на этом устройстве.
3. Вверху страницы появится панель **«Просмотр» / «Редактировать»** — выбери **«Редактировать»**.

**Не открывайте edit-ссылку на чужом компьютере** в общем зале без необходимости. На своём телефоне — можно.

Если панели редактирования нет — см. [FAQ](/ru/dlya-sportsmena/faq#ссылка-для-редактирования-не-работает).

---

## Шаг 3. Заполнить блоки

На странице портфолио — визитка, статистика, достижения, галерея и другие разделы. У каждого блока в режиме редактирования есть кнопка («Редактировать визитку», «Редактировать таблицу» и т.д.).

| Что заполнить | Зачем |
|---------------|--------|
| Полный список блоков | Страница **[Блоки портфолио](/ru/dlya-sportsmena/bloki-portfolio)** |

Минимум перед тем как делиться ссылкой: **имя**, **вид спорта**, хотя бы одно достижение или текст «о себе» — чтобы страница выглядела живой.

Включи **«Подсказки»** на панели сверху — краткие пояснения к разделам.

---

## Шаг 4. Скопировать ссылку для просмотра (до публикации)

Когда заполнено достаточно полей, на панели может появиться **«Скопировать ссылку»** — это адрес **без** `edit=` для предпросмотра.

До **публикации** гости по этой ссылке могут видеть **ограниченный** вид страницы (часть контента скрыта). Владелец с edit-доступом видит всё.

Подробнее: [Что такое ссылка](/ru/dlya-sportsmena/chto-takoe-ssylka).

---

## Шаг 5. Опубликовать профиль

1. Внизу экрана (или в баннере) нажми **«Опубликовать»**, если профиль в статусе **черновик**.
2. Подтверди переход к **оплате** (условия — на странице оплаты; после оплаты публикацию завершает администратор).
3. Пока статус **«Ожидает оплаты»** — страница ещё не считается полностью опубликованной для гостей.
4. После статуса **«Опубликован»** — отправляй **публичную ссылку** (без `edit=`). См. [Сценарии](/ru/dlya-sportsmena/scenarii).

**Заполнение портфолио бесплатно.** Публикация на пилоте может быть **платной** — это отдельный шаг после черновика.

---

## Шаг 6. Поделиться

Скопируй **публичную** ссылку (без `edit=`) и отправь другу, тренеру или в соцсеть — кого выбрал в [сценариях](/ru/dlya-sportsmena/scenarii).

---

## Для родителя

1. Помогите **сохранить edit-ссылку** (заметки, парольный менеджер семьи) — без неё правки только через менеджера.
2. Пройдите шаги 3–5 вместе; объясните разницу: **edit-ссылка** = правка, **обычная ссылка** = просмотр.
3. Решите, **какие данные** показывать (контакты, город, фото).
4. Для 12–17 лет — обсудите, кому можно отправлять ссылку.

---

## Скоро: регистрация на fedat.ru

Целевой продукт — **самостоятельная** регистрация на [fedat.ru](https://fedat.ru), профиль и unlisted-ссылка без заявки на profile.fedat.ru. Когда сценарий будет закрыт на проде, инструкция на этой странице обновится. Пилот через profile.fedat.ru останется рабочим для уже созданных профилей, пока команда не объявит перенос.
